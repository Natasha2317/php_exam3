Описание
Что сделано:
•	аутентификация администратора системы; 
•	формирование администратором новой экспертной сессии и добавление в нее вопросов различного типа; 
•	формирование администратором уникальных ссылок-приглашений для ответа на вопросы сессии; 
•	доступ к вопросам экспертной сессии по одной из ссылок и ввод ответов на вопросы; 
•	возможность у администратора закрыть сессию, после чего ввод ответов на вопросы становится невозможным; 
•	получение сводной информации по ответам на сессию как по всем ответившим, так и по группам экспертов с одинаковой ссылкой (сохраняется и после закрытия сессии); 
•	одновременное проведение любого количества экспертных сессий.
